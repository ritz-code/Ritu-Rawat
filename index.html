PROJECTS

Femme Jolie Clothing

A functioning ecommerce website built with NextJS. The purpose of the project was to ease myself into building a simple ecommerce infrastructure.

The project helped me understand how the backend features like credit card payments are handled(using Stripe) and how NextJs works on the front end.

The server handles authentication, authorization, validation, cart orders, credit card payments and efficiently queries MongoDB.

React Blog

A blog built to get the hang of ReactJS. Another purpose was to start converting my notes into a website to reference, as and when needed. My plan is to keep adding notes as I progress. Learned a lot about JWT, hooks and React routing

The backend consists of a NodeJS server and MySQL database. The server handles authentication, authorization, validation, comment section, errors, and querying the database.

Healthbite App

Healthbite is a nutrition app accessing Nutritionix API. I built it to chart out a healthy diet for myself. The user interface has a search functionality. The search result displays the alories, fat & cholestrol count as well as the nutrition content (Iron, Protein and Fiber content) searched item.

This was one of my earlier projects to get a better understanding of HTML, CSS, JS, and API access, DOM manipulation.
